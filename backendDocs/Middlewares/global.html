volunteerVerification(req, res)

Middleware for verifying the volunteer's JWT token. This function checks for the presence of a token in cookies and verifies it. If verification is successful and the user is found in the database, it responds with the user's username. Otherwise, it returns a status indicating the failure of the token verification.
Parameters:
Name Type Description
req Object The request object from Express, containing cookies with JWT token.
res Object The response object from Express, used to return the verification status and user information.
