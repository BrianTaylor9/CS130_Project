VolunteerLogin(req, res, next)

Manages the volunteer login process. It checks if the email and password match an existing user in the database. If authentication is successful, a unique token is generated for the user and sent as a cookie.
Parameters:
Name Type Description
req Object The request object containing the email and password from the body.
res Object The response object used to send back a success message or an error message.
next function The next middleware function in the stack.

VolunteerSignup(req, res, next)

Handles the volunteer signup process. It creates a new volunteer user if the email does not already exist in the database. A unique token is generated for the user, which is then sent as a cookie.
Parameters:
Name Type Description
req Object The request object containing the email, password, username, and createdAt from the body.
res Object The response object used to send back the created user data or an error message.
next function The next middleware function in the stack.
